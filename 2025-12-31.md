## 1. Multilingual Hidden Prompt Injection Attacks on LLM-Based Academic Reviewing
- **Authors**: Panagiotis Theocharopoulos, Ajinkya Kulkarni, Mathew Magimai. -Doss
- **Link**: https://arxiv.org/abs/2512.23684v1
- **Summary**: 【中文标题】   基于大语言模型的学术评审中的多语言隐藏提示注入攻击    【中文总结】   本研究探讨了大语言模型（LLMs）在学术同行评审流程中面临的文档级隐藏提示注入攻击风险。作者构建了一个包含约500篇ICML录用论文的数据集，通过在其中嵌入四种语言（英语、日语、中文、阿拉伯语）的语义等效对抗指令进行实验。结果表明，除阿拉伯语外，其他语言的提示注入会显著改变LLM生成的评审分数和录用决策，揭示了基于LLM的评审系统存在跨语言差异的安全漏洞。这一发现对高影响力场景下LLM应用的安全性提出了重要警示。

## 2. BOAD: Discovering Hierarchical Software Engineering Agents via Bandit Optimization
- **Authors**: Iris Xu, Guangtao Zeng, Zexue He, Charles Jin, Aldo Pareja, Dan Gutfreund, Chuang Gan, Zhang-Wei Hong
- **Link**: https://arxiv.org/abs/2512.23631v1
- **Summary**: 【中文标题】BOAD：通过多臂老虎机优化发现分层软件工程智能体    【中文总结】   该论文针对大语言模型（LLM）在解决真实世界长周期、分布外软件工程问题时的泛化局限性，提出了一种分层多智能体协作框架BOAD。该方法将复杂任务分解为由协调器管理的子智能体（如定位、编辑、验证等），并通过多臂老虎机模型自动优化子智能体组合，解决传统单体设计中上下文冗余和信用分配难题。实验表明，BOAD在SWE-bench基准测试中超越单智能体和人工设计的多智能体系统，其36B参数模型在最新实时问题上性能优于GPT-4等更大模型，验证了自动发现的分层结构对长周期任务的显著改进效果。

## 3. Divergent-Convergent Thinking in Large Language Models for Creative Problem Generation
- **Authors**: Manh Hung Nguyen, Adish Singla
- **Link**: https://arxiv.org/abs/2512.23601v1
- **Summary**: 【中文标题】大语言模型中发散-收敛思维在创造性问题生成中的应用    【中文总结】该论文针对大语言模型（LLMs）生成教育问题时存在的"人工蜂群思维"效应（即同模型输出相似、跨模型输出同质化）导致问题多样性不足的问题，受Wallas创造力理论和Guilford发散-收敛思维框架启发，提出了CreativeDC双阶段提示方法。该方法通过将创意探索与约束满足分阶段处理，使LLMs能在确定最终问题前探索更广阔的创意空间。实验表明，CreativeDC在保持实用性的同时，生成问题的多样性和新颖性显著优于基线方法，且随着采样量增加，其生成独特问题的有效数量增速更快。

## 4. Toward Trustworthy Agentic AI: A Multimodal Framework for Preventing Prompt Injection Attacks
- **Authors**: Toqeer Ali Syed, Mishal Ateeq Almutairi, Mahmoud Abdel Moaty
- **Link**: https://arxiv.org/abs/2512.23557v1
- **Summary**: 【中文标题】   迈向可信的自主人工智能：一种预防提示注入攻击的多模态框架    【中文总结】   本文提出了一种跨代理多模态溯源感知防御框架，旨在解决由大型语言模型（LLMs）和视觉语言模型（VLMs）驱动的自主AI系统中可能出现的多模态提示注入（PI）攻击问题。该框架通过文本清理代理、视觉清理代理和输出验证代理的协同工作，结合溯源账本记录模态、来源和信任级别元数据，确保代理间通信遵循明确的信任框架，从而防止恶意指令在LangChain或GraphChain式工作流中传播。实验结果表明，该框架显著提高了多模态注入检测的准确性，最小化了跨代理信任泄漏，并增强了自主执行路径的稳定性，为构建安全、可理解和可靠的自主AI系统提供了有效解决方案。

## 5. Lie to Me: Knowledge Graphs for Robust Hallucination Self-Detection in LLMs
- **Authors**: Sahil Kale, Antonio Luca Alfeo
- **Link**: https://arxiv.org/abs/2512.23547v1
- **Summary**: 【中文标题】   骗我试试：基于知识图谱的大语言模型幻觉自检测鲁棒性研究    【中文总结】   本文针对大语言模型（LLMs）生成虚假陈述的“幻觉”问题，提出了一种利用知识图谱提升自检测效果的新方法。该方法通过将模型输出转化为实体-关系知识图谱，并基于图谱结构估算幻觉概率，在GPT-4o和Gemini-2.5-Flash上的实验显示，其检测准确率和F1分数分别比现有最优方法提升16%和20%。研究同时发布了一个增强版人工标注数据集，证实结构化知识表征能有效帮助模型识别原子级事实错误，为构建更安全可靠的LLMs提供了低成本、模型无关的解决方案。
